# gin-web

## 前言
本项目目录参考自 [go-clean-template](!https://github.com/evrone/go-clean-template)

## 目录结构
```shell
config  		- 配置文件
docs    		- 接口文档
internal/server - 服务层
internal/infra  - 存储层
internal/usecase - 业务逻辑层
internal/controller - 控制层
runtime			- 运行时
pkg				- 公共函数包
tests			- 测试文件
schema			- 定义接口数据结构 struct
go.sum
CHANGELOG.md
README.md
```
